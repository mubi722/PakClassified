# Pak Classified

## Overview

Pak Classified is a full-stack online platform for buying, selling, and renting cars. The platform connects vehicle owners with potential buyers and renters through a secure and user-friendly interface.

## Features

* User Authentication and Authorization
* OTP Verification System
* Buy, Sell, and Rent Cars
* Car Listings with Detailed Information
* Interactive Map Location Integration
* Email Notifications using Nodemailer
* Secure User Registration and Login
* Responsive Design
* Search and Filter Functionality

## Tech Stack

### Frontend

* React.js

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Additional Technologies

* Nodemailer
* OTP Verification System
* Map Location Integration

## Project Goals

The goal of Pak Classified is to provide a reliable and efficient platform where users can easily buy, sell, or rent vehicles while ensuring a smooth and secure experience.
